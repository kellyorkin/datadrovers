About the demo. The Discovery Console at datadrovers.com/sawyer runs in fixture mode — pre-captured workspace state that demonstrates the agent's reasoning logic, tool-call patterns, and discovery agenda format without requiring a live MCP connection. The reasoning logic and tool-call patterns are identical to the production version.

About the live runtime. Scope Scout's live runtime — which performs real workspace introspection against the Loomi Connect Marketing MCP — runs in authenticated local environments. On the demo workspace (neon-sock, 123K customers, 1.17M events), Sawyer produced a full discovery agenda including a critical pre-scope finding: all 12 standard taxonomy slots were null despite 27 event types flowing. That finding is the kind of result that changes a client conversation before a single use case is scoped.
